## Get started with Fingertip + Raycast

Raycast is a productivity tool that lets you control your tools with a keyboard. With the Fingertip extension, you can manage your websites directly from Raycast without switching contexts or opening your browser.

![Fingertip Raycast Extension](https://files.raycast.com/148ogfw5bhcsu7xx3kg0tpehp6p6)

![Fingertip Raycast Extension Interface](https://files.raycast.com/gvo2rzezystxibtqgbr4px938bp4)

## Installing the extension

<Steps>
  <Step title="Install Raycast">
    If you haven't already, download and install Raycast from
    [raycast.com](https://www.raycast.com/)
  </Step>
  <Step title="Install the Fingertip extension">
    Click the installation button below:

    <a href="raycast://extensions/mblode/fingertip?source=webstore">
      <img
        src="https://www.raycast.com/mblode/fingertip/install_button@2x.png?v=1.1"
        height="64"
        alt="Install Fingertip Raycast Extension"
        title="Install Fingertip Raycast Extension"
        className="h-16"
      />
    </a>

    Or use this direct link: [Install Fingertip extension](raycast://extensions/mblode/fingertip?source=webstore)

  </Step>
  <Step title="Configure your API key">
    Get your API key from
    [fingertip.com/api-keys](https://fingertip.com/api-keys) and enter it
    in the extension preferences
  </Step>
</Steps>

## Available commands

### Search sites

Browse and search all your Fingertip websites from one place. Quickly access your site dashboard, editor, or view the live site.

### Create site

Quickly set up a new website with custom settings directly from Raycast.
