## Get Started with Fingertip + Make.com

[Make.com](https://make.com) lets you connect Fingertip with thousands of popular apps, automating your work without writing any code. Create powerful visual workflows between Fingertip and your favorite tools to save time and streamline your processes.

## Installation

<Steps>
  <Step title="Connect your Fingertip account">

    1. [Sign up for Make.com](https://www.make.com/en/register) (it's free to
    start)
    2. Log in to your Make.com dashboard
    3. Click "Create a new scenario"
    in the dashboard
    4. Search for and select "Fingertip" as your app 5. When
    prompted, enter your Fingertip API key which you can find at
    [fingertip.com/api-keys](https://fingertip.com/api-keys)

  </Step>
  <Step title="Choose a trigger">

    Select what starts your automation:
    - Choose "Watch Created Form Responses"
    to trigger when someone submits a form
    - Select "Watch Created Site
    Contacts" to trigger when a contact is added
    - Pick "Watch Created Orders"
    to trigger when a new order is placed
    - Select "Watch Created Bookings" to
    trigger when a booking is created
    - Choose "Watch Cancelled Bookings" to
    trigger when a booking is cancelled
    - Pick "Watch Rescheduled Bookings" to
    trigger when a booking is rescheduled

  </Step>
  <Step title="Connect to another app">
    Choose the destination app (like Google Sheets, HubSpot, or any of the
    2,000+ apps available)
  </Step>
  <Step title="Set up the action">
    Configure what happens when the trigger occurs
  </Step>
  <Step title="Test and activate">
    Run a test to verify everything works, then activate your scenario
  </Step>
</Steps>

## Benefits of Connecting Fingertip with Make.com

- **Connect to 2,000+ apps** - Integrate with tools like Google Sheets, OpenAI, Gmail, and more
- **No coding required** - Build automations with a visual workflow interface
- **Free to get started** - No time limit on the Free plan, no credit card required

## Triggers

Events that start a Make.com workflow:

- **Watch Created Form Responses**: Triggers when a new form response is created
- **Watch Created Site Contacts**: Triggers when a new site contact is created
- **Watch Created Orders**: Triggers when a new order is placed
- **Watch Created Bookings**: Triggers when a new booking is created
- **Watch Cancelled Bookings**: Triggers when a booking is cancelled
- **Watch Rescheduled Bookings**: Triggers when a booking is rescheduled

## Actions

Events a scenario performs:

- **Create Site Contact**: Create a new site contact in Fingertip
